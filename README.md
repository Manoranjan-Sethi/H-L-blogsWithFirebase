# Blog App

- [Click here](https://precious-basbousa-c15149.netlify.app) to see preview of our project.

## Tech Stack

- HTML 5
- CSS 3
- Javascript
- React JS
- Firebase
- Firestore
- JSON Server

## Features

- Sign-in
- Create blog
- Edit blog
- Delete blog
- Multiple user accounts login

## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/H-L-blogsWithFirebase.git
```

Run the app

```bash
  npm start
```
